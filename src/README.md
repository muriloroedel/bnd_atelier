# [BND ATELIER]

### Sobre
Este projeto foi desenvolvido para a marcenaria BND Atelier. Foi criada uma página web para captação de novos clientes via web. Agora esta excelente fornecedora de móveis sob medida e modulados pronto tem uma forma de expor seus trabalhos já realizados e criar um showroom online. 
    Agradeço a confiança!
